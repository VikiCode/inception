# Dark-Inception
![](screenshot/logo.png)

A nice sublime text 3 like theme for your Brackets code editor.

  - Sublime like coding interface
  - Easy on eyes

# Installation

Download the theme from extension manager in [Brackets](http://brackets.io/).
